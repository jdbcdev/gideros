NotificationManager:unregisterForPushNotifications

Available since version: Gideros 2011.6

Description:

Unregister device from receiving push notifications. Basically this method invalidates the deviceToken. You still need to remove the token from your server by yourself, or else using invalidated tokens may lead to errors in some push notification services.

Syntax:

NotificationManager:unregisterForPushNotifications()
